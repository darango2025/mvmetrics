Reference

Function Reference

Main wrapper

evaluate_models()

Evaluate multiple models with all metric families A–E. Returns scores, rankings, recommendation, and optional bootstrap CIs.

Metric families

metric_A()

Family A — Weighted marginal sum of RMSE (continuous) + Log-Loss (binary). Marginal baseline, blind to dependence.
metric_B_energy_score()

Family B — Multivariate Energy Score. Strictly proper, sensitive to joint distribution. Requires normalised inputs.
metric_C_variogram_score()

Family C — Variogram Score (p = 0.5). Proper, directly penalises pairwise covariance errors. Robustness check.
metric_D_mdd()

Family D — Marginal-Dependence Decomposition. Separates S_marg from S_dep via PIT correlation matrices.
metric_E_cvwmd() novel

Family E — E-CVWMD. CV-derived variable weights, residual-correlation dependence score, adaptive α* via distance correlation test.

Marginal (univariate) metrics

marginal_rmse()

RMSE per column (named vector).

marginal_mae()

MAE per column (named vector).

marginal_logloss()

Binary cross-entropy Log-Loss for predicted probabilities.

marginal_accuracy()

Classification accuracy at threshold 0.5.

Uncertainty quantification

bootstrap_metrics()

Non-parametric percentile bootstrap for 95% CIs. Pairwise CI overlap test for exactly two models.

Visualisation

plot_metrics()

Four types: bars, forest, heatmap, decomp. Returns a ggplot2 object.

Utilities

normalise_matrix()

Column-wise z-score normalisation using training-set parameters. Required before Metrics B and C.

recommend_metric()

Rule-based metric recommendation. Called automatically by evaluate_models().
